   Как становить:

   2. Найдите мод Immersive Intelligence,
   3. Откройте immersiveintelligence-0.3.1-dev2.jap программой архиваторам ( WinRAR, 7-zip ),
   4. Путь: assets/immersiveintelligence/lang,
   5. Замените файл ru_ru на скачанный.
